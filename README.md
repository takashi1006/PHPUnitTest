# PHPUnit

## 環境

### PHPUnit のバージョン

- PHPUnit 11.3

### 必要な PHP のバージョン

- PHP >= 8.2.0

## 使用方法

1. このリポジトリを「git clone」
1. 「composer install」を実行
   composer、PHP が入っていない場合はインストールする必要がある
1. 「vendor/phpunit/phpunit/phpunit tests」を実行
